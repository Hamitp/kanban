# Akış v0.1.0 — İlk Windows Masaüstü Sürümü

Akış'ın ilk kullanılabilir Windows masaüstü sürümü.

## Öne çıkan özellikler

- Proje bazlı Kanban boardlar
- Özelleştirilebilir sütunlar
- Görev önceliği, etiketler, bekleme nedeni ve kişi atama
- Görev çalışma süresi ve proje ilerleme analitiği
- Etkileşimli mind map
- Arşivleme, geri getirme ve projeler arasında kopyalama
- Dört farklı tema
- `Belgeler\Akış\Save` klasörüne otomatik kayıt
- Saatlik otomatik güvenlik kopyaları
- İnternet bağlantısı olmadan çalışma

## Kurulum

`Akis-Setup-0.1.0-x64.exe` dosyasını indirin ve çift tıklayın. Ayrıntılı anlatım için [Windows kurulum rehberini](https://github.com/Hamitp/kanban/blob/main/docs/KURULUM_REHBERI.md) okuyun.

> Not: Kurulum dosyası henüz ticari kod imzalama sertifikasına sahip değildir. Windows SmartScreen uyarı gösterebilir. Dosyayı yalnızca bu resmi GitHub sürüm sayfasından indirin.
