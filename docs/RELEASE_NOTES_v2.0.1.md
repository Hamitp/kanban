# Akış v2.0.1 — İlk açılış kayıt güvencesi

Bu bakım sürümü, v2.0.0 çalışma alanı dönüşümünün ilk açılışta kullanıcıdan başka bir işlem beklemeden hemen Save dosyasına yazılmasını sağlar.

- Mevcut veriler yine kayıpsız biçimde **Kişisel Alanım** içine taşınır.
- Dönüşüm ilk ekran açılır açılmaz atomik olarak kaydedilir.
- v2.0.0'ın çalışma alanı, arşivleme, geri getirme ve kalıcı silme özellikleri değişmeden korunur.
