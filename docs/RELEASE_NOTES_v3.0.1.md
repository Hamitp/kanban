# Akış v3.0.1 — Kalan iş yükü görünürlüğü

- Genel Bakış özetine **Önceliklendirilmiş** görev göstergesi eklendi.
- Gösterge, önceliklendirilmiş görevlerin açık sayısını doğrudan gösterir.
- Alt açıklamada önceliklendirilmiş ve üzerinde çalışılan görevlerin toplamından oluşan **kalan planlanmış iş** sayısı yer alır.
- Önceliklendirilmemiş havuz bu toplama dahil edilmez; böylece sıraya alınmış gerçek iş yükü ile fikir/görev havuzu birbirine karışmaz.
- Aynı bilgiler İngilizce arayüzde **Prioritized** ve **tasks remaining with active work** olarak sunulur.
